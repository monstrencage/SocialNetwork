Types des objets Facebook

Définitions de types et accesseurs



Ces types sont définis comme des enregistrements, mais on a choisi ici de masquer cela dans l'interface, afin d'obliger à utiliser les accesseurs et d'assurer la généricité du code.
type id = string
Type des identifiants.
type gender =
| Male
| Female
| Unknown (*Genre d'un utilisateur.*)

Informations


type info
Informations prises en charge sur les utilisateurs. Il est important de noter que le genre n'est pas défini sur les pages publiques.
val make_info : string -> Date.t -> gender -> info
val name : info -> string
val birthday : info -> Date.t
val gender : info -> gender
val age : info -> int
